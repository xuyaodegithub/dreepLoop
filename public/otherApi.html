一起火对接API

以下是一起火开放平台说明

项目介绍

一起火为大客户提供API支持，通过API让商家提升发货效率......

对接方式

1.API调用

一起火将给商家分配一个用户，商家将通过此用户获取token，之后API访问以此token最为凭证 ######详细说清请看开发文档 一起火开发文档

2.系统推送

由商家提供http协议的数据接收地址，一起火将数据时时推送到对应的地址，推送内容以JSON格式呈现

####商品信息

  • 合作档口商品上架
名称 类型 说明
productId Long 商品ID
  • 合作档口商品下架
名称 类型 说明
productId Long 商品ID
  • 合作档口商品价格变更通知
名称 类型 说明
productId Long 商品ID
price Double 价格(人民币)

####物流情况

  • 发货通知
名称 类型 说明
orderId Long 订单ID
packageId Long 包裹ID
courierNo String 快递面单号
courierName String 快递公司名称
itemNos String 已发货物编号,逗号隔开
  • 缺货通知
名称 类型 说明
orderId Long 订单ID
productId Long 商品ID
itemNo Long 货物编号
num Int 件数

####售后信息

  • 退货签收通知
名称 类型 说明
orderId Long 订单ID
itemNo Long 货物编号
  • 退货成功或失败通知
名称 类型 说明
orderId Long 订单ID
itemNo Long 货物编号
status Boolean 是否成功
remark String 备注
